image.module installation instructions

1.) Copy this directory to your modules/ directory

2.) Enable the module at: administer -> modules

3.) Configure the module settings at: administer -> settings -> image

4.) Configure cron if you didn't do so already to have your temporary upload
    directory cleaned up


image_gallery.module installation instructions

1.) Enable the image_gallery and taxonomy modules at: administer -> modules

2.) Add image albums for the gallery at: administer -> image galleries


ImageMagick toolkit installation instructions

1.) If ImageMagick is installed, copy image.imagemagic.inc to the
    Drupal's include/ directory

2.) Enable and configure the toolkit at: administer -> settings -> image toolkit
